# Html Table
Simple HTML Table Builder 

##### To use it in your Maven build add:
```xml
<repositories>
	<repository>
	    <id>jitpack.io</id>
	    <url>https://jitpack.io</url>
	</repository>
</repositories>

```

and the dependency:

 ```xml
<dependency>
    <groupId>io.github.atkawa7</groupId>
    <artifactId>htmltable</artifactId>
    <version>1.0.0</version>
 </dependency>
```
